
##GEOspatial Index


###Store an Index
```sh
db.collection.ensureIndex({location:'2D', type:1})
```

###find the closest points
```sh
db.collection.find({location:{$near:[50,50]}})

db.collection.find({location:{$near:[50,50]}}).limit(3) // Get the closest 3
```
